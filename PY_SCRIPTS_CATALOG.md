# Python 脚本目录与作用说明

下表汇总了当前项目中所有 Python 文件的路径与主要作用，方便快速定位代码。

| 路径 | 作用 |
| --- | --- |
| src/config.py | 定义求解器、模型、约束、输出和数据等配置数据类，并提供加载、验证及格式转换工具。 |
| src/data_loader.py | 从 CSV 文件读取模型输入，解析为数据类，构建线路顺序、行程时间映射与班次数等派生结构。 |
| src/data_models.py | 声明线路、站点、换乘区、行程时间、同步对等数据模型以及聚合的 ModelData 容器。 |
| src/utils.py | 提供时间格式化、统计计算、目录创建、配置加载、模型摘要保存等通用工具函数和枚举。 |
| src/bstdt_model.py | 基于输入数据构建 BST-DT 优化模型，创建决策变量并添加行程、同步、周期和容量约束，设置目标函数。 |
| src/check_data_logic.py | 校验同步对是否可行：检查线路是否访问指定换乘区并输出缺失映射或无效同步的提示。 |
| scripts/run_small_instance.py | 运行小规模实例的端到端脚本：加载数据、创建配置、构建并求解模型、保存与分析结果。 |
| scripts/run_medium_instance.py | 运行中等规模实例的脚本：准备目录与配置，加载数据，构建并求解模型并保存结果。 |
| scripts/analyze_results.py | 读取求解结果文件，汇总生成报告与可选图表，对比基线并输出分析摘要。 |
